HR & Employee API Guide

Endpoints for Attendance, Leave Requests, Job Applications, and User Management.

Attendance

GET /api/attendance

Get Today's Attendance for Current User.

POST /api/attendance/check-in
POST /api/attendance/check-out

Leave Requests

GET /api/leaves

Get My Leaves (History).

POST /api/leaves

Submit a new Leave Request.

  • Payload (JSON)
{
  "leaveType": "Sick Leave", // Or Annual, Casual
  "startDate": "2024-03-01",
  "endDate": "2024-03-03",
  "reason": "Flu"
}

Job Applications

POST /api/applications

Public endpoint for submitting job applications.

  • Payload (FormData)
Key Type Required
fullName String Yes
email String Yes
phone String Yes
cv File Yes
position ID Yes

User Management

GET /api/users/profile

Get Current User Profile.

POST /api/users/change-password

Change Password.

GET /api/users/employees

List all Employees (Admin Only).
