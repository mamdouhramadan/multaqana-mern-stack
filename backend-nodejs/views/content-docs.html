Content API Guide

Endpoints for managing dynamic content: News, Magazines, and FAQs. All list endpoints support pagination.

Global Query Params:

?page=1 ?limit=10 ?search=keyword

News

GET /api/news

Retrieve paginated news articles.

POST /api/news

Create a new article. Requires authentication.

  • Payload (FormData)
  • Response
Key Type Required Description
title String Yes Article headline.
content String Yes Full HTML or Text content.
image File Option Cover image (max 5MB).
category ID Yes Category Object ID. 

{
    "success": true,
    "data": {
        "_id": "65b...",
        "title": "Breaking News",
        "slug": "breaking-news",
        "image": "/uploads/news/img-123.jpg"
    }
}

Magazines

GET /api/magazines

List all magazines.

POST /api/magazines

Upload a new magazine PDF.

  • Payload (FormData)
Key Type Required Description
title String Yes Magazine Title.
file File Yes PDF File.
thumbnail File Yes Cover Image.
