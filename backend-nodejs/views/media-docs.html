Media API Guide

Endpoints for managing Videos, Photo Albums, and Files.

Videos

GET /api/videos

List all videos (Paginated).

POST /api/videos

Create Video. Supports File Upload or External URL.

  • Payload (FormData)
Key Type Required Description
title String Yes Video Title.
videoType String Yes 'file' or 'url'.
videoFile File Cond Required if type='file'.
videoUrl String Cond Required if type='url'.
category ID Yes Category Object ID.

Photo Albums

GET /api/albums
POST /api/albums

Create Album with multiple images.

  • Payload (FormData)
Key Type Required Description
title String Yes Album Title.
images Files[] Yes Multiple images (Max 20).
category ID Yes Category Object ID.

Files (Repository)

GET /api/files
POST /api/files
