Structure API Guide

Endpoints for organizational structure: Categories, Departments, and Positions.

Categories

GET /api/categories

Filter by module using query params.


?module=news
POST /api/categories
  • Payload (JSON)
{
  "title": "Technology",
  "description": "Tech news",
  "module": "news" // news, video, faq, file, album
}

GET /api/departments
POST /api/departments

Payload: { "title": "IT", "description": "..." }

Positions

GET /api/positions
POST /api/positions

Payload: { "title": "Senior Dev", "department": "DEPT_ID" }
